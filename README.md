# Hand Camera

Hand tracking and camera-based interaction using OpenCV, MediaPipe, and computer vision.

## Stack

- Python 3.14+
- OpenCV 5.0
- MediaPipe 0.10
- dlib 20.0
- scikit-learn, PyTorch, Flask/FastAPI

## Install

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
